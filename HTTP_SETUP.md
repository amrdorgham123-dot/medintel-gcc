# MedIntel GCC — من HTML إلى HTTP

## السبب الحقيقي وراء مشاكل "الملف مش بيفتح" اللي حصلت قبل كده
`frontend.html` مكتوب عشان يتصل بالـ API بمسارات نسبية (`API = ""`)، ودا معناه إنه
**لازم** يتفتح كرابط HTTP من نفس السيرفر اللي شغّل عليه الباك إند — مش كملف
`file://` بالـ double-click. لو فتحته بالـ double-click، الـ fetch كان هيفشل
حتى لو السيرفر شغّال، لأن المتصفح كان بيدوّر على الداتا جوه نظام الملفات نفسه
مش على `localhost`.

الخبر الحلو: `app.py` أصلاً بيعرض `frontend.html` بنفسه على الرابط الرئيسي —
مش محتاج سيرفر منفصل للفرونت إند.

---

## 1) تشغيل التطبيق كامل محليًا عبر HTTP (الطريقة الصح)

```bash
cd project_9
pip install -r requirements.txt
uvicorn app:app --reload --port 8420
```

بعدين افتح المتصفح على:

```
http://127.0.0.1:8420/
```

**ده رابط HTTP حقيقي**، مش ملف. الداشبورد والبحث والفلاتر والمقارنة كلها هتشتغل
لأن الفرونت إند والباك إند على نفس الأصل (origin).

- `--reload` يخلي السيرفر يعيد التحميل تلقائيًا لو عدّلت كود
- غيّر `--port 8420` لأي بورت تاني لو مشغول

---

## 2) تشغيل النسخة المستقلة عبر HTTP بدل الـ double-click

النسخة المستقلة (`medintel-standalone.html`) شغالة أصلًا بالـ double-click لأنها
مالهاش أي API خارجي، لكن لو عاوزها HTTP فعلاً (مثلاً عشان تفتحها من جهاز تاني
على نفس الشبكة):

```bash
cd project_9
python3 -m http.server 8000
```

وافتح:

```
http://127.0.0.1:8000/medintel-standalone.html
```

أي جهاز تاني على نفس الواي فاي يقدر يفتحها عن طريق IP جهازك بدل 127.0.0.1
(مثلاً `http://192.168.1.5:8000/medintel-standalone.html`).

---

## 3) رفعها أونلاين برابط ثابت (Render.com — مجاني)

أنا مش قادر أرفعها بنفسي من هنا (مفيش عندي وصول لخدمات استضافة)، لكن دي أسرع
طريقة تعملها بنفسك في حوالي 5 دقايق:

1. اعمل حساب مجاني على **render.com**
2. ارفع فولدر `project_9` (اللي فيه `app.py`, `frontend.html`, `medintel.db`,
   `requirements.txt`) على GitHub repo (خاص أو عام)
3. في Render: **New → Web Service** → اربطه بالـ repo
4. الإعدادات:
   - **Build Command:** `pip install -r requirements.txt`
   - **Start Command:** `uvicorn app:app --host 0.0.0.0 --port $PORT`
5. اضغط Deploy — هيديك رابط زي:
   `https://medintel-gcc.onrender.com`

ده رابط ثابت حقيقي أي حد يقدر يفتحه من أي جهاز، من غير ما تشغّل حاجة عندك.

**ملحوظة:** الطبقة المجانية في Render بتنام بعد 15 دقيقة من غير استخدام، وأول
طلب بعد النوم بياخد شوية ثواني يصحّى. لو محتاج يفضل شغال 24/7 من غير نوم،
محتاج خطة مدفوعة (حوالي $7/شهر).
